Ruby Cheat Sheet

Hashes, Arrays, Strings, Integers

Here's a handy cheat sheet to use as a reference to understanding basic Ruby Hashes, Arrays, Strings, Integers!

Hashes
animals ={

'dog' => 'canine',

'cat' => 'feline',

'donkey' => 'asinine'}

end

puts animals['dog'] #=> 'canine' 		A Hash is a dictionary-like collection of unique keys and their values. Also called associative arrays, they are similar to Arrays, but where an Array uses integers as its index, a Hash allows you to use any object type. When you store a value in a hash, you actually supply two objects, the key and the value. You can retrieve the value by using the key. The values in a hash can be objects of any type. The example to the left uses hash literals: a list of key => value pairs between braces.

Arrays
Array = ['lion', 0, nil, 'chicken']

array[0] #=> 'lion'		 Arrays are ordered, integer-indexed collections of any object. Arrays have an order system called an index. the first item of every array starts at 0, then 1, then 2, and so on. A negative index is assumed to be relative to the end of the array. Index of -1 indicates the last element of the array, -2 is the next to last element in the array, and so on.

Strings
"hello"

"Blue"

'CAPS'

'football'
 Strings are the most common object used in Ruby. Strings are an alphanumeric characters that you can identify by the use of single or double quotation marks that wrap around the text.

Integers
10 + 5 = 15

1.5 - 5.5 = -4

2** = 4

3.3 * 4 = 13.2

10 / 2 = 5		 In Ruby there are two classes for numbers, Fixnum are negative and positive whole numbers and Float are decimal numbers. These are the most common types of numbers in Ruby and you can also do mathematical calculations with both.
