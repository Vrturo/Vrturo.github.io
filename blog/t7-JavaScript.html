Ruby vs JavaScript

In this Blog I wanted to go over the main differences I see in Ruby and Javascript functions. The structure can be similar but they are very different languages. Functions in JavaScript are the same as methods in Ruby. The syntax of declaring a function vs. declaring a method is slightly different

Ruby

JavaScript
def plus_10(num = 0)
num + 10
end

plus_10 => #10
plus_10(5) => #15 		function plus_10(num)
{return (num) || 0) + 10; }


plus_10(); // 10
plus_10(5); // 15
In Ruby you define a method using the def function. You can give a method an argument and set the argument value so you can start adding to it. In JavaScript you define a function by using function. You can give functions arguments as well. In order for you to return the value you have to use the return method and have it return whatever value you give it as true using the ||.
